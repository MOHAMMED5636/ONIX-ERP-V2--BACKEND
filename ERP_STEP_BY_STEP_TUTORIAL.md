## ðŸŽ¯ Tutorial 4: Using the Kanban Board

### Steps:

#### Step 1: Open Kanban Board
1. Click **"Tasks"** in menu
2. Click **"Kanban"** tab (or view selector)
3. **Result:** Kanban board appears

#### Step 2: Understand Columns

**You'll see 5 columns:**

1. **Pending** (Left)
   - New tasks
   - Not started yet
   - Color: Gray/Blue

2. **In Progress** (Middle-left)
   - Tasks being worked on
   - Active work
   - Color: Yellow/Orange

3. **Completed** (Middle-right)
   - Finished tasks
   - Done!
   - Color: Green

4. **On Hold** (Right)
   - Paused tasks
   - Waiting for something
   - Color: Orange

5. **Cancelled** (Far right)
   - Cancelled tasks
   - Won't be done
   - Color: Red

#### Step 3: Move a Task

**Example: Start Working on a Task**

1. Find task in **"Pending"** column
2. **Click and hold** on the task card
3. **Drag** it to **"In Progress"** column
4. **Release** mouse button
5. âœ… Task moved! Status updated automatically

**Example: Complete a Task**

1. Find task in **"In Progress"** column
2. **Drag** it to **"Completed"** column
3. **Release**
4. âœ… Task completed!

#### Step 4: View Task Details
1. **Click** on any task card
2. **Result:** Task details panel opens
3. See full information
4. Can edit, add comments, etc.
5. Click outside panel to close
